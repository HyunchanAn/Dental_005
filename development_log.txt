## 2026-05-26 Streamlit Cloud 패키지 의존성 오류(libglib2.0-0, libffi7) 해결
- 오류 원인: Streamlit Cloud의 OS 환경 업데이트(Debian 12) 및 기존 apt 캐시 충돌.
- 해결 방안: 요구사항(requirements.txt)은 이미 opencv-python-headless로 설정되어 있고 packages.txt는 없으므로, Streamlit Cloud 환경 자체 캐시 삭제 및 재부팅(Reboot) 안내.

## 2026-05-26 Streamlit Cloud 모델 가중치 파일 누락 오류(FileNotFoundError) 확인
- 오류 원인: 프로젝트의 .gitignore 파일에 weights/ 폴더와 *.pt 파일이 예외 처리되어 있어 GitHub 저장소에 업로드되지 않음. Streamlit Cloud가 코드를 가져갈 때 가중치 파일이 없어 발생한 문제.
- 조치 계획: 각 가중치 파일의 크기가 144MB로 GitHub 기본 용량 제한(100MB)을 초과하므로, Git LFS(Large File Storage)를 통해 업로드할 수 있도록 설정 변경 및 사용자 확인 요청.

## 2026-05-26 Git LFS 설정 및 모델 가중치 업로드 진행
- 조치 내용: .gitignore 파일에서 weights/ 및 *.pt 예외 규칙을 제거하고, git lfs track "*.pt" 명령어로 LFS 추적을 시작함.
- 대용량 파일(.pt) 3개를 커밋하여 GitHub 저장소에 푸시(Push)함.
- 이후 Streamlit Cloud에서 다시 배포/재부팅 시 정상적으로 모델을 불러와 작동할 예정임.
